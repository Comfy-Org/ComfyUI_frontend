---
question: "支持哪些客户端？"
order: 1
---

对于 Claude Code、Claude Desktop 或 Codex，在任意客户端中将 [https://cloud.comfy.org/mcp](https://cloud.comfy.org/mcp) 添加为自定义连接器或远程 MCP 服务器，然后在提示时登录。

对于不支持 OAuth 的客户端，请使用 Comfy API 密钥连接。将文档 [https://docs.comfy.org/agent-tools/cloud](https://docs.comfy.org/agent-tools/cloud) 发送给你的智能体，它会为你完成安装。
