---
question: "Which clients are supported?"
order: 1
---

For Claude Code, Claude Desktop, or Codex, add [https://cloud.comfy.org/mcp](https://cloud.comfy.org/mcp) as a custom connector or remote MCP server in any client, then sign in when prompted.

For clients that don't support OAuth, connect with a Comfy API key. Send the docs [https://docs.comfy.org/agent-tools/cloud](https://docs.comfy.org/agent-tools/cloud) to your agent and it will figure out the installation for you.
