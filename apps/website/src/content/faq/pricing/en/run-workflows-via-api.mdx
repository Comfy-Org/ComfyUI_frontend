question: "Can I run workflows via API?"
order: 13
---

Yes. Run Comfy workflows programmatically via API. Concurrency limits scale with your plan: 1 / 3 / 5 on Standard / Creator / Pro. It's built for integrating ComfyUI into your apps, automating batch jobs, or running production pipelines. If you need to request increasing limits you can do so [here](https://comfy-org.portal.usepylon.com/forms/team-plan-requests).
